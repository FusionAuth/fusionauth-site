== Overview

https://www.twilio.com/[Twilio] is a popular third party messaging API. This integration when enabled allows Passport
to deliver push messaging during Two Factor authentication.

== Configuration

The Twilio integration may be enabled using the link:../apis/integrations[Integrations] API or through the Passport UI by navigating
to `Settings` -> `Integrations`.
