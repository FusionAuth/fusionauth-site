Now that everything is written, let's test the workflow. Fire up the Node.js application using this command.

```shell script title="Start the Node.js Server"
npm run dev
```

Open `http://localhost:3000/` in a browser.

<img src="/img/quickstarts/changebank-landing.png" alt="The ChangeBank Home Screen" width="1200" role="bottom-cropped" />

Click the `Login` link, this will take us to FusionAuth's login page.

<img src="/img/quickstarts/changebank-login.png" alt="The Login Screen" width="1200" role="bottom-cropped" />

From here, you can log in with the account you created. If you log in with a valid account that is registered for the application, you arrive at the home page, but are greeted by name.

<img src="/img/quickstarts/changebank-loggedin.png" alt="Successful Login" width="1200" role="bottom-cropped" />
