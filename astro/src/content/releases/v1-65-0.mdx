<Item category="new-feature" >
Introduced Tenant Manager SSO, which enables self-service SSO configuration for identity providers configured within tenant manager.
</Item>

<Item category="new-feature" issue="1303">
Introduced a voice option for multi-factor authentication (MFA), allowing users to receive a code via audio instead of text.
</Item>

<Item category="fix" >
Improved the performance of `PATCH` and `PUT` calls to `/api/identity-provider` for deployments with thousands of tenants or applications.
</Item>

<Item category="enhancement" >
Added opt-in retry support with exponential backoff to the Go and Java client libraries.
</Item>
