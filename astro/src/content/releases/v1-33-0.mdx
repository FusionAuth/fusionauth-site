</Item>

</Item>

<Item category="security" issue="1549">

</Item>

<Item category="security" issue="1591">

</Item>

<Item category="breaking-change" issue="1591">
Required obtaining a Trust Token from the Two-Factor Login API to use the Change Password API for users with Two-Factor enabled.
</Item>

</Item>
