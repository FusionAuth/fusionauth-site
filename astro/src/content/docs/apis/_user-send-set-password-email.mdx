import InlineField from 'src/components/InlineField.astro';

  <APIField name="sendSetPasswordEmail" type="Boolean" optional defaults="false">
    If `true`, FusionAuth will send the User an email asking them to set their password. The Email Template that is used is the one specified in the <InlineField>setPasswordEmailTemplateId</InlineField> {props.set_password_field_config_blurb}.

    When you set this value to `true`, any provided <InlineField>password</InlineField> field is ignored.

    If you have also enabled email verification and do not skip verification with the <InlineField>skipVerification</InlineField> parameter, only the setup password email will be sent to the user. Setting up the password using the email sent during this user create operation will verify the User's email if it is not already verified.

    If the SMTP email configuration is not complete, or disabled, this value is ignored.
  </APIField>
