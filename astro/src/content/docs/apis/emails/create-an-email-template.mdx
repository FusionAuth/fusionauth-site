This API is used to create an Email Template. Specifying an Id on the URI will instruct FusionAuth to use that Id when creating the Email Template. Otherwise, FusionAuth will generate an Id for the Email Template.

<TryInExplorer method="POST" path="/api/email/template" tag="email" />

## Request

<API method="POST" uri="/api/email/template" authentication={["global-api-key"]} title="Create an Email Template without providing an Id. An Id will be automatically generated."/>

<API method="POST" uri="/api/email/template/{emailTemplateId}" authentication={["global-api-key"]} title="Create an Email Template with the provided Id"/>

### Request Parameters

<APIBlock>
  <APIField name="emailTemplateId" type="UUID" optional immutable defaults="secure random UUID">
    The Id to use for the new Email Template. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

<EmailTemplateRequestBody />

## Response

The response for this API contains the information for the Email Template that was created.
