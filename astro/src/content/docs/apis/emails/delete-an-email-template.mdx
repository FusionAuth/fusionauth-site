This API is used to delete an Email Template. You must specify the Id of the Email Template on the URI.

<TryInExplorer method="DELETE" path="/api/email/template/{emailTemplateId}" tag="email" />

## Request

<API method="DELETE" uri="/api/email/template/{emailTemplateId}"  authentication={["global-api-key"]} title="Delete an Email Template By Id"/>

### Request Parameters

<APIBlock>
  <APIField name="emailTemplateId" type="UUID" required>
    The Id of the Email Template to delete.
  </APIField>
</APIBlock>

## Response

This API does not return a JSON response body.

<StandardDeleteResponseCodes never_search_error />
