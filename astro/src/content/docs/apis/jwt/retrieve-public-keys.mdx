This API is used to retrieve Public Keys generated by FusionAuth. These can be used to cryptographically verify JWTs signed with the corresponding private key.

<TryInExplorer method="GET" path="/api/jwt/public-key" tag="jwt" />

## Request

<API method="GET" uri="/api/jwt/public-key" authentication={["none"]} title="Retrieve all Public Keys"/>

<XFusionauthTenantIdHeaderScopedOperation />

<API method="GET" uri="/api/jwt/public-key?applicationId={applicationId}" authentication={["none"]} title="Retrieve a single Public Key for a specific Application by Application Id"/>

### Request Parameters

<APIBlock>
  <APIField name="applicationId" type="UUID" optional>
    The Application Id is used to retrieve a specific Public Key. This will return the Public Key that has been specifically configured for the provided Application to sign the access token.

    A public key may not be available for an Application if it is configured to use the global JWT configuration or a HMAC is the configured algorithm for JWT signing.
  </APIField>
</APIBlock>

<XFusionauthTenantIdHeaderScopedOperation />


<Aside type="version">
This API has been available since 1.8.0
</Aside>

<API method="GET" uri="/api/jwt/public-key?kid={kid}" authentication={["none"]} title="Retrieve a single Public Key by Key Identifier"/>

### Request Parameters

<APIBlock>
  <APIField name="kid" optional>
    The Key Id used to retrieve a specific Public Key. This will return the Public Key associated with the Key Id.
  </APIField>
</APIBlock>
