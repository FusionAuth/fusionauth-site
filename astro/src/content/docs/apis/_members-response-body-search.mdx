#### Response Body

<APIBlock>
</APIBlock>


  <APIField name="members" type="Array">
    A list of the Group Members that matched the search criteria.
  </APIField>
  <APIField name="members[x].data" type="Object">
    An object that can hold any information about the Group Member that should be persisted.
  </APIField>
  <APIField name="members[x].groupId" type="UUID">
    The unique Group Id that indicates the Group this membership represents.
  </APIField>
  <APIField name="members[x].id" type="UUID">
    The unique Id of this Group Member.
  </APIField>
  <APIField name="members[x].insertInstant" type="Long">
    The [instant](/docs/reference/data-types#instants) that this membership was created.
  </APIField>
  <APIField name="members[x].userId" type="UUID">
    The unique User Id that represents the User this membership represents.
  </APIField>
  <APIField name="total" type="Integer">
    The total number of Group Members matching the search criteria. Use this value along with the <InlineField>numberOfResults</InlineField> and <InlineField>startRow</InlineField> in the Search request to perform pagination.
  </APIField>
