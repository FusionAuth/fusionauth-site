This API is used to create a new Group.

## Request

<API method="POST" uri="/api/group" authentication={["api-key"]} title="Create a Group with a randomly generated Id"/>

<XFusionauthTenantIdHeaderCreateOperation />

<br/>

<API method="POST" uri="/api/group/{groupId}" authentication={["api-key"]} title="Create a Group with the provided unique Id"/>

### Request Parameters

<APIBlock>
  <APIField name="groupId" type="UUID" optional immutable defaults="secure random UUID">
    The Id to use for the new Group. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

<XFusionauthTenantIdHeaderCreateOperation />

<GroupRequestBody />

## Response

The response for this API contains the Group that was created.
