title: User Registration API
description: Learn about the APIs for creating, retrieving, updating and deleting User Registrations.
---
This page contains the APIs that are used to manage User Registrations. A registration is the association between a User and an Application that they log into. Here are the APIs:

| Operation | Method | Endpoint |
|---|---|---|
| <span id="create-a-user-registration-for-an-existing-user"></span><span id="request"></span><span id="request-parameters"></span><span id="response"></span>[Create a User Registration (for an existing user)](/docs/apis/registrations/create-a-user-registration-for-an-existing-user) | `POST` | `/api/user/registration/{userId}` |
| <span id="create-a-user-and-registration-combined"></span><span id="request-1"></span><span id="request-parameters-1"></span><span id="response-1"></span>[Create a User and Registration (combined)](/docs/apis/registrations/create-a-user-and-registration-combined) | `POST` | `/api/user/registration/{userId}` |
| <span id="retrieve-a-user-registration"></span><span id="request-2"></span><span id="request-parameters-2"></span><span id="response-2"></span>[Retrieve a User Registration](/docs/apis/registrations/retrieve-a-user-registration) | `GET` | `/api/user/registration/{userId}/{applicationId}` |
| <span id="update-a-user-registration"></span><span id="request-3"></span><span id="request-parameters-3"></span><span id="response-3"></span>[Update a User Registration](/docs/apis/registrations/update-a-user-registration) | `PUT` | `/api/user/registration/{userId}/{applicationId}` |
| <span id="delete-a-user-registration"></span><span id="request-4"></span><span id="request-parameters-4"></span><span id="response-4"></span>[Delete a User Registration](/docs/apis/registrations/delete-a-user-registration) | `DELETE` | `/api/user/registration/{userId}/{applicationId}` |
| <span id="verify-a-user-registration"></span><span id="request-5"></span><span id="request-parameters-5"></span><span id="request-body"></span><span id="response-5"></span>[Verify a User Registration](/docs/apis/registrations/verify-a-user-registration) | `POST` | `/api/user/verify-registration`<br />`/api/user/verify-registration/{verificationId}` |
| <span id="resend-a-user-registration-verification-email"></span><span id="request-6"></span><span id="request-parameters-6"></span><span id="request-parameters-7"></span><span id="response-6"></span><span id="response-body"></span>[Resend a User Registration Verification Email](/docs/apis/registrations/resend-a-user-registration-verification-email) | `PUT` | `/api/user/verify-registration?applicationId={applicationId}&email={email}`<br />`/api/user/verify-registration?applicationId={applicationId}&email={email}&sendVerifyRegistrationEmail={sendVerifyRegistrationEmail}` |
