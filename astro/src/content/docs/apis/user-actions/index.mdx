---
title: Manage User Action API
description: Learn about the APIs for creating, retrieving, updating and deleting user actions.
---
This page contains the APIs for managing user actions. This API does not cover actually actioning users. Instead, this is the CRUD API to manage the user action definitions.

If you want to apply an existing user action to a user, see the [Actioning Users API](/docs/apis/actioning-users) and the guide on [how to use User Actions](/docs/lifecycle/manage-users/user-actions).

Here are the APIs:

| Operation | Method | Endpoint |
|---|---|---|
| <span id="create-a-user-action"></span><span id="request"></span><span id="request-parameters"></span><span id="response"></span>[Create a User Action](/docs/apis/user-actions/create-a-user-action) | `POST` | `/api/user-action`<br />`/api/user-action/{userActionId}` |
| <span id="retrieve-a-user-action"></span><span id="request-1"></span><span id="request-parameters-1"></span><span id="response-1"></span>[Retrieve a User Action](/docs/apis/user-actions/retrieve-a-user-action) | `GET` | `/api/user-action`<br />`/api/user-action/{userActionId}` |
| <span id="update-a-user-action"></span><span id="request-2"></span><span id="request-parameters-2"></span><span id="response-2"></span>[Update a User Action](/docs/apis/user-actions/update-a-user-action) | `PUT` | `/api/user-action/{userActionId}` |
| <span id="delete-a-user-action"></span><span id="request-3"></span><span id="request-parameters-3"></span><span id="response-3"></span>[Delete a User Action](/docs/apis/user-actions/delete-a-user-action) | `DELETE` | `/api/user-action/{userActionId}`<br />`/api/user-action/{userActionId}&hardDelete=true` |
| <span id="reactivate-a-user-action"></span><span id="request-4"></span><span id="request-parameters-4"></span><span id="response-4"></span>[Reactivate a User Action](/docs/apis/user-actions/reactivate-a-user-action) | `PUT` | `/api/user-action/{userActionId}?reactivate=true` |
