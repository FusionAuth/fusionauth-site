This API is used to create a new IP ACL.

## Request

<API method="POST" uri="/api/ip-acl" authentication={["global-api-key"]} title="Create an IP ACL with a randomly generated Id"/>

<API method="POST" uri="/api/ip-acl/{ipAccessControlListId}" authentication={["global-api-key"]} title="Create an IP ACL with the provided unique Id"/>

### Request Parameters

<APIBlock>
  <APIField name="ipAccessControlListId" type="UUID" optional defaults="secure random UUID">
    The Id to use for the new IP ACL. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

## Response

The response for this API contains the IP ACL that was created.
