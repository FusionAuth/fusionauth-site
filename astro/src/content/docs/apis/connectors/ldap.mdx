## Overview

<Aside type="version">
This API has been available since 1.18.0
</Aside>

The following APIs are provided to manage LDAP Connectors.

## Create the LDAP Connector

### Request

<APIAuthentication icon="lock">Create a LDAP Connector with a randomly generated Id.</APIAuthentication>
<APIURI method="POST">/api/connector</APIURI>

<APIAuthentication icon="lock">Create a LDAP Connector with the provided unique Id.</APIAuthentication>
<APIURI method="POST">/api/connector/`{connectorId}`</APIURI>

The <InlineField>type</InlineField> in the request JSON is used to determine that you are creating a LDAP Connector.

#### Request Parameters

<APIBlock>
  <APIField name="connectorId" type="UUID" optional defaults="secure random UUID">
    The Id to use for the new Connector. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

<LdapRequestBody />

### Response

<StandardPostResponseCodes never_search_error />

<LdapResponseBody />

## Retrieve the LDAP Connector

### Request

<APIAuthentication icon="lock">Retrieve the LDAP Connector by Id</APIAuthentication>
<APIURI method="GET">/api/connector/`{connectorId}`</APIURI>

#### Request Parameters

<APIBlock>
  <APIField name="connectorId" type="UUID" required>
    The Id of the Connector to retrieve.
  </APIField>
</APIBlock>

### Response

<StandardGetResponseCodes never_search_error />

<LdapResponseBody />

## Delete the LDAP Connector

### Request

<APIAuthentication icon="lock">Delete the LDAP Connector by Id</APIAuthentication>
<APIURI method="DELETE">/api/connector/`{connectorId}`</APIURI>

<APIBlock>
  <APIField name="connectorId" type="UUID" required>
    The Id of the Connector to delete.
  </APIField>
</APIBlock>

### Response

This API does not return a JSON response body.
