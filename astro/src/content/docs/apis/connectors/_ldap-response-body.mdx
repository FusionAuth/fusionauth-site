#### Response Body

<APIBlock>
  <APIField name="connector.authenticationURL" type="String">
    The fully qualified LDAP URL to authenticate.
  </APIField>
  <APIField name="connector.baseStructure" type="String">
    The top of the LDAP directory hierarchy. Typically this contains the `dc` (domain component) element.
  </APIField>
  <APIField name="connector.connectTimeout" type="Integer">
    The connect timeout for the HTTP connection, in milliseconds. Value must be greater than `0`.
  </APIField>
  <APIField name="connector.data" type="Object">
    An object that can hold any information about the Connector that should be persisted.
  </APIField>
  <APIField name="connector.debug" type="Boolean">
    Determines if debug should be enabled to create an event log to assist in debugging integration errors.
  </APIField>
  <APIField name="connector.id" type="UUID">
    The unique Id of the LDAP Connector.
  </APIField>
  <APIField name="connector.identifyingAttribute" type="String">
    The entry attribute name which is the first component of the distinguished name of entries in the directory.
  </APIField>
  <APIField name="connector.insertInstant" type="Long">
    The [instant](/docs/reference/data-types#instants) when the LDAP Connector was created.
  </APIField>
  <APIField name="connector.lambdaConfiguration.reconcileId" type="UUID">
    The Id of an existing [Lambda](/docs/apis/lambdas). The lambda is executed after the user authenticates with the connector. This lambda can create a user, registrations, and group memberships in FusionAuth based on attributes returned from the connector.
  </APIField>
  <APIField name="connector.lastUpdateInstant" type="Long">
    The [instant](/docs/reference/data-types#instants) when the LDAP Connector was last updated.
  </APIField>
  <APIField name="connector.loginIdAttribute" type="String">
    The entity attribute name which stores the identifier that is used for logging the user in.
  </APIField>
  <APIField name="connector.name" type="String">
    The unique LDAP Connector name.
  </APIField>
  <APIField name="connector.readTimeout" type="Integer">
    The read timeout for the HTTP connection, in milliseconds. Value must be greater than `0`.
  </APIField>
  <APIField name="connector.requestedAttributes">
    The attributes to request for an entry from the DIT (Directory Information Tree).
  </APIField>
  <APIField name="connector.securityMethod" type="String">
    The LDAP security method.
    Possible values are:
    * `None` - Requests will be made without encryption.
    * `LDAPS` - A secure connection will be made to a secure port over using the LDAPS protocol.
    * `StartTLS` - An un-secured connection will initially be established, followed by secure connection established using the StartTLS extension.
  </APIField>
  <APIField name="connector.systemAccountDN" type="String">
    The distinguished name of an entry that has read access to the directory.
  </APIField>
  <APIField name="connector.systemAccountPassword" type="String">
    The password of an entry that has read access to the directory.
  </APIField>
  <APIField name="connector.type" type="String">
    This field must be set to `LDAP`.
  </APIField>
</APIBlock>
