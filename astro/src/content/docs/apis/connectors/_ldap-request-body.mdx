#### Request Body

<APIBlock>
  <APIField name="connector.authenticationURL" type="String" required>
    The fully qualified LDAP URL to authenticate.
  </APIField>
  <APIField name="connector.baseStructure" type="String" required>
    The top of the LDAP directory hierarchy. Typically this contains the `dc` (domain component) element.
  </APIField>
  <APIField name="connector.connectTimeout" type="Integer" required>
    The connect timeout for the HTTP connection, in milliseconds. Value must be greater than `0`.
  </APIField>
  <APIField name="connector.data" type="Object" optional>
    An object that can hold any information about the Connector that should be persisted.
  </APIField>
  <APIField name="connector.debug" type="Boolean" optional defaults="false">
    Determines if debug should be enabled to create an event log to assist in debugging integration errors.
  </APIField>
  <APIField name="connector.identifyingAttribute" type="String" required>
    The entry attribute name which is the first component of the distinguished name of entries in the directory.
  </APIField>
  <APIField name="connector.lambdaConfiguration.reconcileId" type="UUID" required>
    The Id of an existing [Lambda](/docs/apis/lambdas). The lambda is executed after the user authenticates with the connector. This lambda can create a user, registrations, and group memberships in FusionAuth based on attributes returned from the connector.
  </APIField>
  <APIField name="connector.loginIdAttribute" type="String" required>
    The entity attribute name which stores the identifier that is used for logging the user in.
  </APIField>
  <APIField name="connector.name" type="String" required>
    The unique LDAP Connector name.
  </APIField>
  <APIField name="connector.readTimeout" type="Integer" required>
    The read timeout for the HTTP connection, in milliseconds. Value must be greater than `0`.
  </APIField>
  <APIField name="connector.requestedAttributes" required>
    The attributes to request for an entry from the DIT (Directory Information Tree).
  </APIField>
  <APIField name="connector.securityMethod" type="String" required>
    The LDAP security method.
    Possible values are:
    * <InlineFieldValue>None</InlineFieldValue> - Requests will be made without encryption.
    * <InlineFieldValue>LDAPS</InlineFieldValue> - A secure connection will be made to a secure port over using the LDAPS protocol.
    * <InlineFieldValue>StartTLS</InlineFieldValue> - An un-secured connection will initially be established, followed by secure connection established using the StartTLS extension.
  </APIField>
  <APIField name="connector.systemAccountDN" type="String" required>
    The distinguished name of an entry that has read access to the directory.
  </APIField>
  <APIField name="connector.systemAccountPassword" type="String" required>
    The password of an entry that has read access to the directory.
  </APIField>
  <APIField name="connector.type" type="String" required>
    This field must be set to `LDAP`.
  </APIField>
</APIBlock>
