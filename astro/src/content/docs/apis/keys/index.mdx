<Aside type="version">
Available since `1.6.0`
</Aside>

Use this API to view, create, and update keys, certificates, and secret values (see the [Key Master documentation](/docs/operate/secure/key-master) for an overview of this functionality).

Use the following APIs to manage Keys:

| Operation | Method | Endpoint |
|---|---|---|
| <span id="retrieve-a-key"></span><span id="request"></span><span id="request-parameters"></span><span id="response"></span>[Retrieve a Key](/docs/apis/keys/retrieve-a-key) | `GET` | `/api/key`<br />`/api/key/{keyId}` |
| <span id="update-a-key"></span><span id="request-parameters-1"></span><span id="response-1"></span>[Update a Key](/docs/apis/keys/update-a-key) | `PUT` | `/api/key/{keyId}` |
| <span id="delete-a-key"></span><span id="request-parameters-2"></span><span id="response-2"></span>[Delete a Key](/docs/apis/keys/delete-a-key) | `DELETE` | `/api/key/{keyId}` |
| <span id="search-for-keys"></span><span id="request-1"></span><span id="request-parameters-3"></span><span id="request-body"></span><span id="response-3"></span>[Search for Keys](/docs/apis/keys/search-for-keys) | `GET`<br />`POST` | `/api/key/search?algorithm=RS256&name={name}&type=RSA`<br />`/api/key/search` |
| <span id="generate-a-key"></span><span id="request-parameters-4"></span><span id="response-4"></span>[Generate a Key](/docs/apis/keys/generate-a-key) | `POST` | `/api/key/generate`<br />`/api/key/generate/{keyId}` |
| <span id="import-a-key"></span><span id="request-parameters-5"></span><span id="response-5"></span>[Import a Key](/docs/apis/keys/import-a-key) | `POST` | `/api/key/import`<br />`/api/key/import/{keyId}` |
