This API is used to generate a new Key.

<API method="POST" uri="/api/key/generate" authentication={["global-api-key"]} title="Generate a new Key with a randomly generated Id"/>

<API method="POST" uri="/api/key/generate/{keyId}" authentication={["global-api-key"]} title="Generate a new Key with the provided unique Id"/>

## Request Parameters

<APIBlock>
  <APIField name="keyId" type="UUID" optional defaults="secure random UUID">
    The Id to use for the new key. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

<KeyGeneratePostRequestBody />

## Response

The response for this API contains the Key that was generated.
