#### Response Body

<APIBlock>
  <APIField name="message.textMessage" type="String">
    The SMS message text. Present when <InlineField>message.type</InlineField> is `SMS`.
  </APIField>
  <APIField name="message.message" type="String" since="1.65.0">
    The voice message text. Present when <InlineField>message.type</InlineField> is `Voice`.
  </APIField>
  <APIField name="message.type" type="String">
    The type of the message.

    Supported values include:

    * `SMS`
    * `Voice` <InlineField since="1.65.0" />
  </APIField>
  <APIField name="email.errors">
    An [Errors](/docs/apis/errors) object that contains any errors in the Message Template.
  </APIField>
</APIBlock>
