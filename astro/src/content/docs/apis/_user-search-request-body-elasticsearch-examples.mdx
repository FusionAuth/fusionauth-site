###### Search Using A Query

Searches using `query` have two parts. You must build the JSON <InlineField>query</InlineField> parameter, then escape it and include it in a request.

A request includes not only a query, but also additional search parameters, such as sorting or pagination. For more, see the [Searching Users With Elasticsearch](/docs/lifecycle/manage-users/search/user-search-with-elasticsearch) guide.
