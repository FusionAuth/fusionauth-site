### Request Body

<APIBlock>
  <APIField name="ipAccessControlList.entries" type="Array" required>
    A list of IP ranges and the action to apply for each.  One and only one entry must have a <InlineField>startIPAddress</InlineField> of `*` to indicate the default action of the IP ACL.
  </APIField>
  <APIField name="ipAccessControlList.entries[x].action" type="String" required>
    The action to take for this IP Range. The possible values are:

    * `Allow` - allow all IPs in the range
    * `Block` - block all IPs in the range

  </APIField>
  <APIField name="ipAccessControlList.entries[x].endIPAddress" type="String" required>
    The ending IP (IPv4) for this range.  The only time this is not required is when <InlineField>startIPAddress</InlineField> is equal to `*`, in which case this field is ignored. This value must be greater than or equal to the <InlineField>startIPAddress</InlineField>. To define a range of a single IP address, set this field equal to the value for <InlineField>startIPAddress</InlineField>.
  </APIField>
  <APIField name="ipAccessControlList.entries[x].startIPAddress" type="String" required>
    The starting IP (IPv4) for this range.
  </APIField>
  <APIField name="ipAccessControlList.name" type="String" required>
    The unique name of this IP ACL.
  </APIField>
</APIBlock>
