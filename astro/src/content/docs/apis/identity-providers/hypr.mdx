### Request

<APIURI method="POST">/api/identity-provider</APIURI>

## Retrieve the HYPR Identity Provider

There is only one HYPR Identity Provider, so this Identity Provider may be retrieved by type or Id.

### Request

<APIAuthentication icon="lock">Retrieve the HYPR Identity Provider by type</APIAuthentication>
<APIURI method="GET">/api/identity-provider`?type=HYPR`</APIURI>

<APIAuthentication icon="lock">Retrieve the HYPR Identity Provider by Id</APIAuthentication>
<APIURI method="GET">/api/identity-provider/`778985b7-6fd8-414d-acf2-94f18fb7c7e0`</APIURI>

### Response

<StandardGetResponseCodes never_search_error />

<HyprResponseBody idp_display_name={idp_display_name} idp_since={idp_since} idp_linking_strategy={id_linking_strategy} />

<APIAuthentication icon="lock">Delete the HYPR Identity Provider by type</APIAuthentication>
<APIURI method="DELETE">/api/identity-provider`?type=HYPR`</APIURI>

<APIAuthentication icon="lock">Delete the HYPR Identity Provider by Id</APIAuthentication>
<APIURI method="DELETE">/api/identity-provider/`778985b7-6fd8-414d-acf2-94f18fb7c7e0`</APIURI>

## Start the HYPR Login Request

This API is used to initiate a HYPR login request when integrating without the FusionAuth hosted login pages.

### Request

<APIAuthentication icon="unlock">Start a login request</APIAuthentication>
<APIURI method="POST">/api/identity-provider/start</APIURI>

<IdentityProviderStartRequestBody idp_display_name={idp_display_name} idp_since={idp_since} idp_linking_strategy={id_linking_strategy} />

### Response

The response for this API contains a code that can be used to complete the login request.

<IdentityProviderStartResponseBody idp_display_name={idp_display_name} idp_since={idp_since} idp_linking_strategy={id_linking_strategy} />
