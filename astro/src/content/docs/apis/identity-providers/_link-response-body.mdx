#### Response Body
The response body for this API contains either a single Link or an array of Links. When you call this API with <InlineField>identityProviderId</InlineField>, <InlineField>identityProviderUserId</InlineField>, and optionally the <InlineField>userId</InlineField>, the response will contain a single Link. When you call this API with <InlineField>userId</InlineField> and either <InlineField>identityProviderId</InlineField> or <InlineField>identityProviderUserId</InlineField>, the response will contain an array of Links.
