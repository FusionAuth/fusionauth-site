<APIField name="verificationIds" type="Array<String>" optional since="1.62.0">
    An array of strings generated to create a user with identities that have already been verified.

    <Aside type="note">
        These `verificationIds`:
        * Must be created using the [Start Identity Verification API](/docs/apis/identity-verify#start-identity-verification) with an <InlineField>existingUserStrategy</InlineField> of `MustNotExist`
        * Must have already had completed verification using the [Complete Identity Verification API](/docs/apis/identity-verify#complete-identity-verification)
        * Take precedence over the values in <InlineField>user.email</InlineField> and/or <InlineField>user.phoneNumber</InlineField>. The values in those fields must match those represented by the `verificationIds`.
    </Aside>
</APIField>
