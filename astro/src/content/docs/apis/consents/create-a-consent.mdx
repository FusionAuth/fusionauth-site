This API is used to create a new Consent.


## Request

<API method="POST" uri="/api/consent" authentication={["global-api-key"]} title="Create a new Consent with a randomly generated Id"/>

<API method="POST" uri="/api/consent/{consentId}" authentication={["global-api-key"]} title="Create a new Consent with the provided unique Id"/>

### Request Parameters

<APIBlock>
  <APIField name="consentId" type="UUID" optional immutable defaults="secure random UUID">
    The Id to use for the new Consent. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

## Response

The response for this API contains the Consent that was created.
