This API is used to retrieve Recovery Codes for a User.

<TryInExplorer method="GET" path="/api/user/two-factor/recovery-code/{userId}" />

## Request

<API method="GET" uri="/api/user/two-factor/recovery-code/{userId}" authentication={["api-key"]} title="Retrieve Recovery Codes (deprecated)"/>

<DeprecatedSince since="1.64.0">This endpoint is deprecated. To provide access to a user's recovery codes, [generate new recovery codes](/docs/apis/two-factor/generate-recovery-codes) and deliver those</DeprecatedSince>

### Request Parameters

<APIBlock>
  <APIField name="userId" type="UUID" required>
    The Id of the User to retrieve Recovery Codes for.
  </APIField>
</APIBlock>

## Response

The response for this API contains the remaining Recovery Codes that are assigned to the User. Each time one is used it is removed, so this response will contain between 0 and 10 codes.

<StandardPostResponseCodes never_search_error />

### Response Body

<APIBlock>
  <APIField name="recoveryCodes" type="Array<String>">
    The array of Recovery Codes.
  </APIField>
</APIBlock>
