Starts an multi-factor request. This would be used for only step up auth, such as when sensitive data is requested.

If you want to provide your own code and/or deliver the code out of band using your own delivery mechanism, this is the right API call. Do not combine this with a [Send a Multi-Factor Code During Login or Step Up](/docs/apis/two-factor/send-a-multi-factor-code-during-login-or-step-up) call, as calling that API will invalidate all other codes associated with the **twoFactorId**, including any you provide.

To require additional factors during login, [Enable Multi-Factor](/docs/apis/two-factor/enable-multi-factor) for the User. Then FusionAuth will handle MFA code collection automatically, if you are using the hosted login pages, or send a status code in response to the login API if you are not.
