## Overview

<Aside type="version">
  This API has been available since 1.50.0
</Aside>

This page contains the APIs for managing the OAuth Scopes of an Application.

## Create a Scope

This API is used to create a Scope for an Application. Specifying an Id on the URI will instruct FusionAuth to use that Id when creating the Scope. Otherwise, FusionAuth will generate an Id for the Scope.

### Request

<API method="POST" uri="/api/application/{applicationId}/scope" authentication={["api-key"]} title="Create a Scope with a randomly generated Id"/>

<API method="POST" uri="/api/application/{applicationId}/scope/{scopeId}" authentication={["api-key"]} title="Create a Scope with a provided unique Id"/>

<XFusionauthTenantIdHeaderScopedOperation />

#### Request Parameters

<APIBlock>
  <APIField name="applicationId" type="UUID" required>
    The Id of the Application.
  </APIField>
  <APIField name="scopeId" type="UUID" optional defaults="secure random UUID">
    The Id to use for the new scope. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

<ScopeRequestBody />

### Response

The response for this API contains the information for the Scope that was created.

<StandardPostResponseCodes />

<ScopeResponseBody />

## Retrieve a Scope

This API is used to retrieve a single Scope for an Application by unique Id.

### Request

<API method="GET" uri="/api/application/{applicationId}/scope/{scopeId}" authentication={["api-key"]} title="Retrieve a Scope by Id"/>

<XFusionauthTenantIdHeaderScopedOperation />

#### Request Parameters

<APIBlock>
  <APIField name="applicationId" type="UUID" required>
    The Id of the Application.
  </APIField>
  <APIField name="scopeId" type="UUID" optional defaults="secure random UUID">
    The Id of the Scope to retrieve.
  </APIField>
</APIBlock>

### Response

The response for this API contains a single Scope.

<StandardGetResponseCodes never_search_error />

<ScopeResponseBody />

## Delete a Scope

This API is used to permanently delete a Scope.

### Request

<API method="DELETE" uri="/api/application/{applicationId}/scope/{scopeId}" authentication={["api-key"]} title="Delete a Scope by Id"/>

#### Request Parameters

<APIBlock>
  <APIField name="applicationId" type="UUID" required>
    The Id of the Application.
  </APIField>
  <APIField name="scopeId" type="UUID" optional defaults="secure random UUID">
    The Id of the Scope to retrieve.
  </APIField>
</APIBlock>

<XFusionauthTenantIdHeaderScopedOperation />

### Response

This API does not return a JSON response body.
