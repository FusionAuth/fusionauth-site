## Overview

<Aside type="version">
This API has been available since 1.26.0
</Aside>

The following APIs are provided to manage Twilio Messengers.

### Operations

## Create the Twilio Messenger

### Request

<APIAuthentication icon="lock">Create a Twilio Messenger with a randomly generated Id.</APIAuthentication>
<APIURI method="POST">/api/messenger</APIURI>

<APIAuthentication icon="lock">Create a Twilio Messenger with the provided unique Id.</APIAuthentication>
<APIURI method="POST">/api/messenger/`{messengerId}`</APIURI>

The <InlineField>type</InlineField> in the request JSON is used to determine that you are creating the Twilio Messenger.

#### Request Parameters

<APIBlock>
  <APIField name="messengerId" type="UUID" optional defaults="secure random UUID">
    The Id to use for the new Messenger. If not specified a secure random UUID will be generated.
  </APIField>
</APIBlock>

<TwilioRequestBody />

### Response

<StandardPostResponseCodes never_search_error />

<TwilioResponseBody />

## Retrieve the Twilio Messenger

### Request

<APIAuthentication icon="lock">Retrieve the Twilio Messenger by Id</APIAuthentication>
<APIURI method="GET">/api/messenger/`{messengerId}`</APIURI>

#### Request Parameters

<APIBlock>
  <APIField name="messengerId" type="UUID" required>
    The Id of the Messenger to retrieve.
  </APIField>
</APIBlock>

### Response

<StandardGetResponseCodes never_search_error />

<TwilioResponseBody />

### Request

<APIAuthentication icon="lock">Delete the Twilio Messenger by Id</APIAuthentication>
<APIURI method="DELETE">/api/messenger/`{messengerId}`</APIURI>

<APIBlock>
  <APIField name="messengerId" type="UUID" required>
