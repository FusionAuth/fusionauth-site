#### Response Body

<APIBlock>
  <APIField name="code" type="String">
    A valid Multi-Factor verification code. 
  </APIField>
  <APIField name="methods" type="Array">
    A list of configured authentication methods. This field will be omitted if none have been configured.
  </APIField>
  <APIField name="methods[x].id" type="String">
    The identifier for this method.
  </APIField>
  <APIField name="methods[x].lastUsed" type="Boolean">
    `true` if this method was used most recently.
  </APIField>
  <APIField name="methods[x].method" type="String">
    The type of this method. There will also be an object with the same value containing additional information about this method.  The possible values are:

    * `authenticator`
    * `email`
    * `sms`
  </APIField>
  <APIField name="methods[x].authenticator" type="Object">
    An object with additional configuration for TOTP authentication methods. Only present if <InlineField>methods``[x]``.method</InlineField> is `authenticator`. 
  </APIField>
  <APIField name="methods[x].authenticator.algorithm" type="String">
    The algorithm used by the TOTP authenticator. With the current implementation, this will always be `HmacSHA1`. The possible values are:

    * `HmacSHA1`
    * `HmacSHA256`
    * `HmacSHA512`
  </APIField>
  <APIField name="methods[x].authenticator.codeLength" type="Integer">
    The length of code generated by the TOTP. With the current implementation, this will always be 6.
  </APIField>
  <APIField name="methods[x].authenticator.timeStep" type="Integer">
    The time-step size in seconds. With the current implementation, this will always be 30.
  </APIField>
  <APIField name="methods[x].email" type="String">
    The value of the email address for this method. Only present if <InlineField>methods``[x]``.method</InlineField> is `email`. 
  </APIField>
  <APIField name="methods[x].mobilePhone" type="String">
    The value of the mobile phone for this method. Only present if <InlineField>methods``[x]``.method</InlineField> is `sms`. 
  </APIField>
  <APIField name="twoFactorId" type="String">
    A value to be provided to the Login endpoint to complete Multi-Factor login or the Send endpoint to send a code.
  </APIField>
</APIBlock>
