#### Request Body

<APIBlock>
  <APIField name="scope.name" type="String" required>
    The name of the Scope. This value will be used in OAuth workflows to request scopes and in tokens to indicate consented scopes.
  </APIField>
  <APIField name="scope.data" type="Object" optional>
    An object that can hold any information about the Scope that should be persisted.
  </APIField>
  <APIField name="scope.description" type="String" optional>
    The description for the Scope.
  </APIField>
  <APIField name="scope.defaultConsentMessage" type="String" optional>
    The default message shown when prompting a user for consent.
  </APIField>
  <APIField name="scope.defaultConsentDetail" type="String" optional>
    The default detail shown when prompting a user for consent.
  </APIField>
  <APIField name="scope.required" type="Boolean" optional defaults="false">
    Whether the scope is required. Users must consent to required scopes to complete the OAuth workflow.
  </APIField>
</APIBlock>
