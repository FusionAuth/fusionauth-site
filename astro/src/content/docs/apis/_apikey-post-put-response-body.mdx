#### Response Body

<APIBlock>
  <APIField name="apiKey.expirationInstant" type="Long" since="1.55.0">
    The expiration [instant](/docs/reference/data-types#instants) of this API key. Using an expired API key for [API Authentication](/docs/apis/authentication) will result in a 401 response code.
  </APIField>
  <APIField name="apiKey.id" type="UUID" renderif={props.apikey_create_response}>
    The Id of the API key.
  </APIField>
  <APIField name="apiKey.id" type="UUID" renderif={props.apikey_update_response}>
    The Id of the API key.
  </APIField>
  <APIField name="apiKey.insertInstant" type="Long">
    The [instant](/docs/reference/data-types#instants) when the API key was added to the FusionAuth database.
  </APIField>
  <APIField name="apiKey.ipAccessControlListId" type="UUID" since="1.30.0">
    The Id of the [IP Access Control List](/docs/apis/ip-acl) limiting access to this API key.
  </APIField>
  <APIField name="apiKey.key" type="String" renderif={props.apikey_create_response}>
    The API key value, this is the string value you will use in the Authorization header to [authenticate API](/docs/apis/authentication) requests.

    If the <InlineField>apiKey.retrievable</InlineField> value is <InlineFieldValue>false</InlineFieldValue> then this will be the **only** time the key is returned in the response.
  </APIField>
  <APIField name="apiKey.key" type="String" renderif={props.apikey_update_response}>
    The API key value. This is the string value you will use in the Authorization header to [authenticate API](/docs/apis/authentication) requests.

    If the <InlineField>apiKey.retrievable</InlineField> value is <InlineFieldValue>false</InlineFieldValue> then this field will not be returned in the response.
  </APIField>
  <APIField name="apiKey.keyManager" type="Boolean">
    Indicates whether this key is a key manager. Setting this flag to true allows this key to be used to call these APIs. An attempt to call these APIs with a non-manager key (`keyManager` set to <InlineFieldValue>false</InlineFieldValue>) will always return a HTTP response status code 401.
  </APIField>
  <APIField name="apiKey.lastUpdateInstant" type="Long">
    The [instant](/docs/reference/data-types#instants) when the API key was last updated in the FusionAuth database.
  </APIField>
  <APIField name="apiKey.metaData.attributes.description" type="String">
    Description of the key.
  </APIField>
  <APIField name="apiKey.name" type="String" since="1.56.0">
    The name of the API key.
  </APIField>
  <APIField name="apiKey.permissions.endpoints" type="Object">
    Endpoint permissions for this key. Each key of the object is an endpoint, with the value being an array of the HTTP methods which can be used against the endpoint. An empty permissions object mean that this is a super key that authorizes this key for all the endpoints.
  </APIField>
  <APIField name="apiKey.retrievable" type="Boolean" since="1.56.0">
    Indicates whether this key is retrievable.

    If this value is <InlineFieldValue>false</InlineFieldValue>, the <InlineField>apiKey.key</InlineField> will not be returned in the API response.
  </APIField>
  <APIField name="apiKey.tenantId" type="String">
    The unique Id of the Tenant. This value is read-only once the key is created.
  </APIField>
</APIBlock>
