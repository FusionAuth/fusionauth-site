title: Login API
description: API documentation for the FusionAuth Login API.
---

This page contains all of the APIs for managing user login and login records.

| Operation | Method | Endpoint |
|---|---|---|
| [Authenticate a User](/docs/apis/login/authenticate) | `POST` | `/api/login` |
| [Authenticate a User with a One Time Password](/docs/apis/login/authenticate-otp) | `POST` | `/api/login` |
| [Complete Multi-Factor Authentication](/docs/apis/login/complete-mfa) | `POST` | `/api/two-factor/login` |
| [Update Login Instant](/docs/apis/login/update-instant) | `PUT` | `/api/login/{userId}/{applicationId}` |
| [Logout a User](/docs/apis/login/logout) | `POST` | `/api/logout` |
| [Search Login Records](/docs/apis/login/search) | `GET` / `POST` | `/api/system/login-record/search` |
| [Export Login Records](/docs/apis/login/export) | `GET` / `POST` | `/api/system/login-record/export` |
