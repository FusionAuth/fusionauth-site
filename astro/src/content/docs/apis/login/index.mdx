This page contains all of the APIs for managing user login and login records.

Use the following APIs to manage Login:

| Operation | Method | Endpoint |
|---|---|---|
| <span id="authenticate-a-user"></span><span id="request"></span>[Authenticate a User](/docs/apis/login/authenticate) | `POST` | `/api/login` |
| <span id="authenticate-a-user-with-a-one-time-password"></span>[Authenticate a User with a One Time Password](/docs/apis/login/authenticate-otp) | `POST` | `/api/login` |
| <span id="complete-multi-factor-authentication"></span>[Complete Multi-Factor Authentication](/docs/apis/login/complete-mfa) | `POST` | `/api/two-factor/login` |
| <span id="update-login-instant"></span>[Update Login Instant](/docs/apis/login/update-instant) | `PUT` | `/api/login/{userId}/{applicationId}` |
| <span id="logout-a-user"></span>[Logout a User](/docs/apis/login/logout) | `POST` | `/api/logout` |
| <span id="search-login-records"></span>[Search Login Records](/docs/apis/login/search) | `GET` / `POST` | `/api/system/login-record/search` |
| <span id="export-login-records"></span><span id="request-6"></span>[Export Login Records](/docs/apis/login/export) | `GET` / `POST` | `/api/system/login-record/export` |
