#### Request Body

<APIBlock>
  <APIField name="action.actionerUserId" type="UUID" required>
    The Id of the User that is taking the action on the User.
  </APIField>
  <APIField name="action.comment" type="String" optional>
    The comment left by the actioner.
  </APIField>
  <APIField name="action.emailUser" type="Boolean" optional defaults="false">
    Whether FusionAuth should send an email to the User.
  </APIField>
  <APIField name="action.expiry" type="Long" optional>
    The expiration [instant](/docs/reference/data-types#instants) of this User Action. This is required for time-based User Actions.
  </APIField>
  <APIField name="action.notifyUser" type="Boolean" optional defaults="false">
    The **notifyUser** flag that is passed along in any events FusionAuth sends to registered Webhooks.
  </APIField>
  <APIField name="broadcast" type="Boolean" optional defaults="false">
    Whether or not FusionAuth will broadcast the User Action to any registered Webhooks.
  </APIField>
</APIBlock>
