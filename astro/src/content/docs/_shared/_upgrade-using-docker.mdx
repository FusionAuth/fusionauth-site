To upgrade FusionAuth when running with `docker-compose`:

. Stop the instance: `docker-compose down`.
. Modify the `docker-compose.yml` file to point to the version of FusionAuth you want. You can see [available tags](https://hub.docker.com/r/fusionauth/fusionauth-app).
. Start it up: `docker-compose up`.
. Log in to the administrative UI.
