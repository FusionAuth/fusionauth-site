Once you arrive in the FusionAuth Admin UI, the first thing you need to do is create an Application. 

If you see a First Time Setup wizard in the main content window, you can click <Breadcrumb>Skip it</Breadcrumb> as you'll be creating things outside of this wizard.

An Application is something that a user can log into. This is the application you are building or that you are migrating to use FusionAuth.

<Steps>

1. Click the <Breadcrumb>Applications</Breadcrumb> menu option on the left side of the page.

   ![Dashboard to Applications](/img/docs/extend/examples/5-minute-intro/dashboard-applications.png)

1. Click the <Breadcrumb>green plus</Breadcrumb> button (the add button) at the top of the page:

   ![Application Listing](/img/docs/extend/examples/5-minute-intro/application-listing.png)

1. On the Add Application form, provide a name for your Application (this is only used for display purposes).

1. Click the <Breadcrumb>OAuth</Breadcrumb> tab. You'll see the following form:

   ![Application Form](/img/docs/extend/examples/5-minute-intro/application-form.png)

   Start with a simple setup that allows existing users to log into your application. There is no need to add any roles or registration configuration.
   Most of the defaults will work, but provide the following items:

   * an **authorized redirect URL** of `http://localhost:3000/oauth-redirect`. This is the route/controller in your application's backend that will complete the OAuth grant and obtain a token. You will see example code below for this route. Note that when you type or paste values into this text field you also must click the popup that appears. Otherwise the value will not be saved.
   * a **Logout URL** of `http://localhost:3000/logout`. This is where the user will be redirected to after they are logged out of FusionAuth's OAuth front-end: your application. You'll see how this is used below.
   * under <Breadcrumb>Enabled Grants</Breadcrumb>, select the **Authorization Code grant**.
   * set the **Require registration** field to `true`

1. Click the blue <Breadcrumb>Save</Breadcrumb> button (disk) at the top of the page.

   ![Save Button Application Form](/img/docs/extend/examples/5-minute-intro/application-form-save.png)

1. Next to your application name, click the <Breadcrumb>green magnifying glass</Breadcrumb>. Scroll down to the <Breadcrumb>OAuth Configuration</Breadcrumb> section and copy the **Client Id** and **Client Secret** values off to a text file. You'll use them later.

  ![Application Details](/img/docs/extend/examples/5-minute-intro/application-details.png)

</Steps>
