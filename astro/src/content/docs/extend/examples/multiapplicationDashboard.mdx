## Make The App

```sh
docker ps rm fa faDb
docker compose down -v
docker compose up

docker run --rm -v ".:/app" -w "/app" node:23-alpine3.19 sh -c  "npm install"

docker run --rm --network host  -v ".:/app" -w "/app" -e "PORT=3001" node:23-alpine3.19 sh -c  "npm run start"
```

## Make dashboard

Index page of FA available at http://localhost:9011.
To turn this into app dashboard, go to http://localhost:9011/admin/theme.
Preview the Bank theme (created in kickstart.json) and select the Index page (fourth item in the list on the left).
Edit the theme.
