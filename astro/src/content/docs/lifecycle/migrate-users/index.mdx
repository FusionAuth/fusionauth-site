## Overview

Oftentimes you are migrating to FusionAuth from another solution, whether it be homegrown or another vendor.

These guides and resources will help you understand and scope the migration process.

## Migration Guides

* [General](/docs/lifecycle/migrate-users/bulk/general) - general migration guidance, strategies and help
* [Auth0](/docs/lifecycle/migrate-users/bulk/auth0) - how to migrate from Auth0 to FusionAuth
* [Microsoft Azure AD B2C](/docs/lifecycle/migrate-users/bulk/azureadb2c) - how to migrate from Microsoft Azure AD B2C to FusionAuth
* [Cognito](/docs/lifecycle/migrate-users/bulk/cognito) - how to migrate from Amazon Cognito to FusionAuth
* [Duende IdentityServer](/docs/lifecycle/migrate-users/bulk/duende) - how to migrate from Duende IdentityServer to FusionAuth
* [Firebase](/docs/lifecycle/migrate-users/bulk/firebase) - how to migrate from Firebase to FusionAuth
* [Keycloak](/docs/lifecycle/migrate-users/bulk/keycloak) - how to migrate from Keycloak to FusionAuth
* [Supabase](/docs/lifecycle/migrate-users/bulk/supabase) - how to migrate from Supabase to FusionAuth
* [Tutorial](/docs/lifecycle/migrate-users/bulk/tutorial) - how to migrate from a single user database to FusionAuth

## Other Useful Migration Resources

* [Plugins](/docs/extend/code/password-hashes/) - custom password hashing support, so that you can migrate without requiring your users to change their password
* [APIs](/docs/apis/) - Migrate data and configuration using the APIs
* [Client libraries](/docs/sdks/) - use the language of your choice to make FusionAuth API calls
* [Connectors](/docs/lifecycle/migrate-users/connectors/) - implement a gradual migration using this feature
* [Import scripts](https://github.com/FusionAuth/fusionauth-import-scripts) - open source scripts to import from CSV, Auth0 and more
* [FusionAuth Editions](/pricing) - some editions include FusionAuth team support for data migration

## Further Assistance

If you need further assistance migrating to FusionAuth, please ask a question in the <a href="/community/forum/" target="_blank">FusionAuth forum</a>.

If you have a licensed edition you may open a <a href="https://account.fusionauth.io/account/support/" target="_blank">support request from your account</a>.
