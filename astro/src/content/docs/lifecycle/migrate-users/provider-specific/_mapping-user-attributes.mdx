The attributes of the User object in FusionAuth are [well documented](/docs/apis/users).

If there is an attribute in your {props.migration_source_name} user which cannot be directly mapped to a FusionAuth attribute, you can place it in the <InlineField>user.data</InlineField> field. This field can store arbitrary JSON values and will be indexed and searchable.
