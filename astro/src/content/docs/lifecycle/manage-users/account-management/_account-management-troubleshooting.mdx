Not able to get your account management page up and running?  Below are common issues and how to fix them.

### Not Registered

Please note, if you are not registered for an application, you will not be able to manage your account information from these self service account management.

From the admin UI click on <strong>Users -> Manage -> Registrations Tab -> Add registration -> Add User</strong>.  Another option is to be programmatically added to the application through an API call.

<img src="/img/docs/lifecycle/manage-users/account-management/account-home-not-registered.png" alt="Login Screen"  width="1200"  role="bottom-cropped" />

### Registered But Configuration Needed

If you see this, you have to select a form to use.

<img src="/img/docs/lifecycle/manage-users/account-management/configuration-incomplete-account.png" alt="Login Screen"  width="1200"  role="bottom-cropped" />

To set a self service form, click under <strong>Applications-> Edit Application -> Registration</strong>

<img src="/img/docs/lifecycle/manage-users/account-management/add-form-account.png" alt="Edit Application Screen - Registrations"  width="1200" />

Select the form you would like to use. A default form is shipped with FusionAuth.
