## Add a Two Factor Method - Authenticator

The Authenticator method is enabled by default on every tenant. To toggle please navigate to <strong>Tenants -> Edit Tenant -> Multi-Factor</strong>. The authenticator method is also referred to as Google Authenticator or Time-Based One-Time Password (TOTP).

### Enable MFA method on Tenant (Admin Facing)

The authenticator factor is enabled by default at the tenant level.  It can be toggled on and off as needed.

<img src="/img/docs/lifecycle/manage-users/account-management/toggle-authenticator-tenant.png" alt="Toggle Authenticator On Tenant"  width="1200" />

### Enable Authenticator Factor from Account Management (User Facing)

<img src="/img/docs/lifecycle/manage-users/account-management/account-management-index-crop.png" alt="Account Management Index"  width="1200"  role="bottom-cropped" />

1. Navigate back to your account page.
2. Click `Manage two-factor`
3. Click `Add two-factor`
4. There will be an option for an `Authenticator app`.

<img src="/img/docs/lifecycle/manage-users/account-management/add-factors-home-crop.png" alt="Add Factors"  width="1200"  role="bottom-cropped" />
<img src="/img/docs/lifecycle/manage-users/account-management/authenticator-add-account.png" alt="Authenticator Add Account"  width="1200"  role="bottom-cropped" />

Next,

1. Scan the QR code with your favorite authenticator application such as Google Authenticator or Authy.
2. Enter the code given.
3. Click the `Enable` button.

### Recovery Codes (User Facing)

Now you will be presented with recovery codes. Save these in a safe space.

<img src="/img/docs/lifecycle/manage-users/account-management/mfa-recovery-codes-crop.png" alt="Recovery Codes"  width="1200"  role="bottom-cropped" />

<Icon name="trophy" /> Success!

Upon the next login, you will be prompted for a code displayed by the Authenticator App in addition to your password.

### See It in Action (User Facing)

With the Authenticator method enabled, if you log out and log back in you will be presented with the following screen in addition to the typical login screen.

<img src="/img/docs/lifecycle/manage-users/account-management/auth-challenge-account-screen-crop.png" alt="Challenge Account Management"  width="1200"  role="bottom-cropped" />
