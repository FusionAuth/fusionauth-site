Use this page to configure Tenant Manager in the FusionAuth admin UI.

Tenant administrators use these settings in the [Tenant Manager App](/docs/lifecycle/manage-users/tenant-manager).

## Tenant Manager Settings Page

Open <Breadcrumb>Settings -> Tenant Manager</Breadcrumb>.

![Tenant Manager Settings](/img/docs/lifecycle/manage-users/tenant-manager/tenant-manager-settings.png)

### Branding

The Branding section includes the `Brand name` field.

* `Brand name` - Sets the Tenant Manager brand name. Maximum length: 255 characters.

### Single Sign-On

The Single Sign-On section includes settings used by tenant-managed identity providers.

* `Registration form` - Select the registration-type form used to determine available attribute mapping targets in Tenant Manager.
* `Enabled universal applications` - Select one or more universal applications that Tenant Manager identity providers can be enabled for.

Tenant Manager identity providers can only be used with selected universal applications.

### Allowed Identity Provider Types

The Allowed Identity Provider Types table lists configured identity provider types, such as OpenID Connect and SAML v2.

From this table, you can:

* edit a configured type
* delete a configured type
* click <InlineUIElement>Add allowed Identity Provider</InlineUIElement> to create a new allowed type
