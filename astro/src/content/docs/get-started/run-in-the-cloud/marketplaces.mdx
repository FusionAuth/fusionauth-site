## Marketplace Install

FusionAuth is available in the following cloud marketplaces:

* [DigitalOcean](https://marketplace.digitalocean.com/apps/fusionauth)
* [AWS](https://aws.amazon.com/marketplace/seller-profile?id=30ecc80d-f96e-4259-ae1c-fc3b994f90ea)

## Paid Features

Once you install FusionAuth using the marketplace, you can [add a license](/docs/get-started/core-concepts/licensing). Doing so will allow you to use [additional features](/docs/get-started/core-concepts/editions-features).
