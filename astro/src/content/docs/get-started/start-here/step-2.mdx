This doc will explain how to get the example application up and running and show them the login button.

## Get the example app

```shell
git clone github.com:fusionauth/fusionauth-example-app.git
```

Now start the application:

```shell
npm install
npm run
```

Open your browser to http://localhost:3000 and you'll see the example application. Click the login button and you'll be taken over to FusionAuth's OAuth login page. Yay!
