FusionAuth releases come in three flavors: major, minor, and patch.

You can tell what a release is from the rightmost non-zero number in the version: `<major>.<minor>.<patch>`. For example:

* `1.0.0`: major release, may significantly change the API
* `1.63.0`: minor release, may make small changes to the API
* `1.45.1`: patch release, never changes the API

For more information about upgrading your instance of FusionAuth, see our [Upgrade FusionAuth Guide](/docs/operate/deploy/upgrade).
