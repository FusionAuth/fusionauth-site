Use the following commands to create a new instance of FusionAuth, complete with all the dependencies you need to run it:

<Steps>
1. Install the prerequisites:

   * On macOS and Windows, one of the following container management tools:
     * [Docker desktop](https://www.docker.com/products/docker-desktop/)
     * [OrbStack](https://docs.orbstack.dev/quick-start) (to use Orbstack for `docker compose` commands after install, run `docker context use orbstack`)
     * [Podman](https://podman.io/docs/installation) (in the commands below, replace `docker` with `podman`)
   * On Linux, [Docker](https://docs.docker.com/get-started/get-docker/) 23 or later

1. Install the FusionAuth CLI:

   ```console
   npm install @fusionauth/cli
   ```

1. Run the following command to create a Docker container that contains a FusionAuth instance, a database, an API key, and an application:

   ```console
   npx fusionauth kickstart:install
   ```

1. Start the Docker container:

   ```console
   npx fusionauth kickstart:start
   ```

1. Visit [http://localhost:9011](http://localhost:9011) to access the Admin UI.
1. Congratulations, you have a working instance of FusionAuth! To set up login for an application, see our [First Time Setup guide](/docs/get-started/download-and-install/setup).
</Steps>
