Copy and paste each command below into a terminal to install using [Homebrew](https://brew.sh/).
You'll also need to manually install a database for FusionAuth to work.

<Steps>

1. Tap the FusionAuth package:

   ```console
   brew tap fusionauth/homebrew-fusionauth</code>
   ```
1. Install the FusionAuth tap:

   ```console
   brew install fusionauth-app</code>
   ```

1. Start the FusionAuth service:
   
   ```console
   brew services start fusionauth-app</code>
   ```

1. Visit [http://localhost:9011](http://localhost:9011) to open the Admin UI.

</Steps>
