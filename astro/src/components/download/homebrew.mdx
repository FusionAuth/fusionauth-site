Copy and paste each command below into a terminal to install using [Homebrew](https://brew.sh/).
You'll also need to manually install a database for FusionAuth to work.

<Steps>

1. Tap the FusionAuth tap:

   ```console
   brew tap fusionauth/homebrew-fusionauth
   ```

1. Trust the tap:
 
   ```console
   brew trust fusionauth/fusionauth
   ```

1. Install the FusionAuth app:

   ```console
   brew install fusionauth-app
   ```

1. Start the FusionAuth service:
   
   ```console
   brew services start fusionauth-app
   ```

1. [Install and configure a database](/docs/get-started/download-and-install/database).

1. Visit [http://localhost:9011/admin](http://localhost:9011/admin) to access the Admin UI.

1. Congratulations, you have a working instance of FusionAuth! To set up login for an application, see our [First Time Setup guide](/docs/get-started/download-and-install/setup).

</Steps>
