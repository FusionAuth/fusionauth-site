Identity, Authentication & Authorization

Learn everything you need to know about authentication, authorization, identity, and access management from our team of industry experts.

Entities (permissions) allow you to model relationships and enable machine-to-machine authentication using the Client Credentials grant.
Protecting APIs with JSON Web Tokens

FusionAuth handles user authentication in your serverless and microservices based applications. Leverage popular API gateways to protect your APIs.
Consolidate Users’ Data

Integrating single sign-on into your applications means your users can access all their applications with one set of credentials, such as a username and password.
Migrating to FusionAuth

Oftentimes you are migrating to FusionAuth from another solution, whether it be homegrown or another vendor.
User Registration & Profile Management

FusionAuth self-service account management allows users to manage their accounts, including the ability to add Multi-factor Authentication (MFA).
