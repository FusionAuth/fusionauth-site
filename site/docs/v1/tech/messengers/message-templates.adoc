== Adjust a Message Template

A message template allows you to customize how your messenger communicates with your audience.  You can add custom styles and text using FreeMarker.

A new template can be added by click on the icon:plus[type=fas, role=ui-button green] Add button

image::messengers/message-templates-home.png[Message Templates Home , width=1200]

=== Add Message Template

image::messengers/message-templates-add.png[Message Templates Add, width=1200]

==== Form Fields

[.api]
[field]#Id# [optional]#Optional#::
A unique UUID is auto generated if left blank.

[field]#Name# [required]#Required#::
For display only. Name of the messenge template.

[field]#Type# [optional]#Optional#::
The type of message template.  Options include: SMS, Kafka, and Generic.

[field]#Default Text# [required]#Required#::
The input accepts FreeMarker inmput and will allow crafting a custom template.

=== Add Message Template - Localization

image::messengers/message-templates-add-locals.png[Message Templates Add Locals, width=1200]

==== Form Fields

[.api]
[field]#Id# [optional]#Optional#::
You can enter localization options here.  See our guide to localization link:/docs/v1/tech/themes/localization[here]

Click `icon:plus[type=fas] Add localization` and fill additional details as outlined below.

=== Add Message Template - Localization Details

image::messengers/message-templates-add-locals-popup.png[Message Templates Add Locals Popup, width=1200]

==== Form Fields

[.api]
[field]#Locale# [optional]#Optional#::
Enter the language you are wishing to localize.

[field]#Template# [optional]#Optional#::
Fill in localization messaging here.

Check results by clicking `icon:search[type=fas] Preview`
