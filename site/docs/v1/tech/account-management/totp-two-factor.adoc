=== Add a Two Factor Method - TOTP

The authenticator is the only service enabled by default. To fully enable:

image::account-management/authenticator-add-account.png[Authenicator Add Account, width=1200]

1. Scan the QR code with your favorite TOTP-based application.
2. Enter the code given
3. Click the `icon:save[type="fas"] Enable` button

Now you will be presented with recovery codes.  Save these in a safe space.

image::account-management/mfa-recovery-codes-crop.png[Recovery Codes, width=1200]

icon:trophy[] Success! +

Upon the next login, the user will be prompted for a code displayed by an Authenticator App in addition to your password.

==== See it in action

With TOPT 2FA enabled, if you log out and log back in you will be presented with the following screen:

image::account-management/auth-challenge-account-screen-crop.png[Challenge Account Management, width=1200]
