[field]#{base_field_name}.algorithm# [type]#[Long]#::
The identifier for the signing algorithm used with the passkey. These values are defined by https://www.rfc-editor.org/rfc/rfc9053[RFC 9053]. FusionAuth supports a subset of these algorithms.

[field]#{base_field_name}.attestationType# [type]#[String]#::
The type of attestation provided when the passkey was registered. Supported values include:
 * `none` - no attestation was provided
+
FusionAuth does not currently support attestation. The only valid value at this time is `none`.
