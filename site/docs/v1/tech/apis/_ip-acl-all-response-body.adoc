==== Response Body

[.api]
[field]#ipAccessControlLists# [type]#[Array]#::
The list of all IP ACLs

[field]#ipAccessControlList``[x]``.entries# [type]#[Array]#::
A list of IP ranges and the action to apply for each.  One and only one entry will have a `startIPAddress` of `*` to indicate the default action of the IP ACL.

[field]#ipAccessControlList``[x]``.entries``[y]``.action# [type]#[String]#::
The action to take for this IP Range.

[field]#ipAccessControlList``[x]``.entries``[y]``.endIPAddress# [type]#[String]#::
The ending IP (IPv4) for this range.

[field]#ipAccessControlList``[x]``.entries``[y]``.startIPAddress# [type]#[String]#::
The starting IP (IPv4) for this range.

[field]#ipAccessControlList``[x]``.name# [type]#[String]#::
The name of this IP ACL.
