== Overview

An IP ACL (Access Control List) is a list of IP ranges that are either allowed or blocked.
Along with one `*` or everything range that defines the default behavior.

== Retrieve an IP ACL

This API is used to retrieve a single IP ACL by unique Id or all of the configured IP ACLs.

=== Request

[.api-authentication]
link:/docs/v1/tech/apis/authentication#api-key-authentication[icon:lock[type=fas]] Retrieve all of the IP ACLs
[.endpoint]
.URI
--
[method]#GET# [uri]#/api/ip-acl#
--

[.api-authentication]
link:/docs/v1/tech/apis/authentication#api-key-authentication[icon:lock[type=fas]] Retrieve an IP ACL by Id
[.endpoint]
.URI
--
[method]#GET# [uri]#/api/ip-acl/`{ipAclId}`#
--

==== Request Parameters

[.api]
[field]#ipAclId# [type]#[UUID]# [required]#Required#::
The unique Id of the IP ACL to retrieve.

=== Response

The response for this API contains either a single IP ACL or all of the IP ACLs. When you call this API with an Id the response will
contain a single IP ACL. When you call this API without an Id the response will contain all of the IP ACLs. Both response types are
defined below along with an example JSON response.
