== Overview

[NOTE.since]
====
This API has been available since 1.26.0
====

The FusionAuth APIs are primarily secured using API keys. This page describes APIs that are used to manage API keys. Here are the APIs.

* <<Retrieve an API Key>>
* <<Create an API Key>>
* <<Generate a New API Key From an Exising API Key>>
* <<Update an API Key>>
* <<Delete an API Key>>

== Retrieve an API Key

This API is used to retrieve a single API Key by unique Id.


=== Request

[.api-authentication]
link:/docs/v1/tech/apis/authentication#api-key-authentication[icon:lock[type=fas]] Retrieve an API Key by Id
[.endpoint]
.URI
--
[method]#GET# [uri]#/api/api-key/`\{APIKeyId\}`#
--

==== Request Parameters

[.api]
[field]#APIKeyId# [type]#[UUID]# [required]#Required#::
The unique Id of the API Key to retrieve.

The response for this API contains a single AIP Key. The response is defined below along with an example JSON response.

== Generate a New API Key From an Exising API Key

This API is used to generate a new API key based on an existing API Key. If the key that is being copied must not be set an key manager, in which case, this API will return an error.

[.api-authentication]
link:/docs/v1/tech/apis/authentication#api-key-authentication[icon:lock[type=fas]] Update the Key with the given Id
[.endpoint]
.URI
--
[method]#PUT# [uri]#/api/api-key/#
--

.Example API Key request JSON
[source,json]
{
  "sourceKeyId": "48b5762e-6829-4d7e-b53d-f7db5f5d3d1f"
}


=== Response

The response for this API contains the Key that was created.
