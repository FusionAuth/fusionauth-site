==== Response Body

[.api]
[field]#ipAccessControlList.entries# [type]#[Array]#::
A list of IP ranges and the action to apply for each.  One and only one entry will have a `startIPAddress` of `*` to indicate the default action of the IP ACL.

[field]#ipAccessControlList.entries``[x]``.action# [type]#[String]#::
The action to take for this IP Range.

[field]#ipAccessControlList.entries``[x]``.endIPAddress# [type]#[String]#::
The ending IP (IPv4) for this range.

[field]#ipAccessControlList.entries``[x]``.startIPAddress# [type]#[String]#::
The starting IP (IPv4) for this range.

[field]#ipAccessControlList.name# [type]#[String]#::
The name of this IP ACL.
