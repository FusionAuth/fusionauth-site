==== Response Body

[.api]
[field]#identityProviderLink.displayName# [type]#[String]#::
The name of the link when initially created. This name is often set to the username or email returned by the identity provider once a user authenticates for the first time. For instance, a link with `displayName` of `Erlich's Link` initially might change to `erlich-bachman@gmail.com` after their first login with the Identity Provider.
+
It's important to note that this `displayName` value may not necessarily bet set to their username or email used to authenticate with the third-party identity provider.
[field]#identityProviderLink.identityProviderId# [type]#[UUID]#::
The unique Id of the identity provider.

[field]#identityProviderLink.identityProviderUserId# [type]#[String]#::
The Id for the User that is provided by the identity provider.

[field]#identityProviderLink.userId# [type]#[UUID]#::
The FusionAuth User Id that is linked to the identity provider.
