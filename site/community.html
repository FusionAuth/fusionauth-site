Community & Support

With hundreds of thousands of downloads in the last year, we've quickly developed an active worldwide community of developers sharing challenges and solutions every day. Visit the sites below to contribute your knowledge and help guide FusionAuth's future.

Full API Docs

Dive into our full API documentation for specific details, code examples, and tutorials on everything FusionAuth.

Ask Questions

Stuck trying to get something to work? Jump over to StackOverflow to post questions in the community. Make sure you watch the FusionAuth tag - you could have the answer to someone else’s question.

Report Bugs

Found a bug or want to request a new feature? Jump over to GitHub and review the list of current issues and feature requests. We also have several active bounties posted where you can earn cash for contributing to FusionAuth.

Chat with users

Looking to get even more involved? We’ve got a Slack group for anyone using FusionAuth to ask questions and help other community members. Everyone is welcome, so if you are interested in joining, click the button to send us an join request email.

Write Tutorials

Want to show off your auth coding chops to the community? We are always paying $$$ for posts and tutorials that will help other FusionAuth users. Write a post with code examples or create a video tutorial, it’s up to you. Find out more details and get started here.

Need more support?

We offer full support plans for both self-hosted and managed hosting packages.
