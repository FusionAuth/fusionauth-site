FusionAuth Reactor

FusionAuth Reactor™ is a powerful suite of features developed to extend FusionAuth's core functionality and is available to customers with a paid edition of FusionAuth.

START FREE TRIAL

Breached Password Detection

FusionAuth will check user login credentials against a set of leaked databases that we've collected from all over the Internet, even the hard to reach places. You can enable Breached Password Detection during account creation, password resets, password changes and login. You can then force the user to change their password or notify them of the breach via email.

Advanced Registration Forms

NEW!

You can build custom forms and perform complex validation during registration without writing any code. Our Advanced Registration Forms provides you the ability to collect custom user data during registration and store it within FusionAuth.

LEARN MORE     

LDAP Integration

NEW!

Want to connect external data sources such as LDAP or a custom API to FusionAuth? Now you can with Connectors. When you configure a Connector, the rules for how a user is authenticated are completely customizable. You can also configure user data migration rules on a per-Connector basis.
