Connectors

Connect FusionAuth to LDAP or API data sources. Then, authenticate or migrate users contained in that datastore.

START FREE TRIAL

Supports LDAP and APIs

Connectors securely communicate with LDAP directories, such as Active Directory. They can also connect FusionAuth with custom HTTP APIs, allowing authentication against homegrown user datastores without a massive data migration.

Migrate or authenticate

Once a user is authenticated against a connected datasource, you can migrate them to FusionAuth with simple configuration. Alternatively, leave their data untouched and have FusionAuth transparently delegate future auth decisions.

Segment your users

Apply a Connector to users matching a certain domain or domains. Doing so ensures only users with matching email addresses will ever be authenticated against the datasource, while still letting the user enjoy the FusionAuth login experience.
