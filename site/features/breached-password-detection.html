Breached Password Detection

Prevents user from using compromised credentials to sign up or log in.

GET STARTED FOR FREE

FusionAuth checks user login credentials against a set of leaked databases that we've collected from all over the Internet, even the hard to reach places. You can enable Breached Password Detection during account creation, password resets, password changes and login. You can then force the user to change their password or notify them of the breach via email.

Receive a webhook event for every breached password found, allowing you to trigger external actions such as disabling access or notifying an internal team. FusionAuth also marks the user's account as using a breached password until it is changed, useful for reporting and analytics.
