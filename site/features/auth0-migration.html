Migrate from Auth0 to FusionAuth

Auth0 and FusionAuth both provide authorization, authentication and user management. Migrate to FusionAuth for more control and less money.

GET STARTED FOR FREE

Test FusionAuth

The community edition of FusionAuth can be downloaded and run for free. Test out integrations with other identity providers, including unlimited OIDC and SAML connections. Make sure it works for your needs.

Export your data

You'll need to export your users, password hashes, and other configuration from Auth0. With the password hash export, your users will never know you changed auth services. Pull other configuration from Auth0 by using the management APIs.

Import your data

Import your user and password data using provided scripts. Migrate other entities such as Connections, Roles and Rules manually. Or, if you have a lot of configuration, script the migration using the Auth0 and FusionAuth APIs.

Create a tenant in FusionAuth

Create a tenant in FusionAuth. Tenants, applications and users are all core concepts. Tenants are unlimited, so create as many as you want.

Export user data, then import it

Install the Export extension and download your user data. Load your users using provided scripts, and use the FusionAuth APIs to move over other Auth0 configuration.

Set up applications in FusionAuth

An application is anything you can log in to. Set them up in FusionAuth, then modify your applications to integrate with FusionAuth using SAML, OIDC, or other OAuth grants.
