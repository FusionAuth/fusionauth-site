Features

Here are just a few of the awesome features you get for free with FusionAuth.

On-Premise or Private Cloud

FusionAuth is designed to give businesses complete control over their data resides, and easily installs in dev, staging, or production.

Always Single-Tenant

Unlike mass-market solutions, FusionAuth is always single-tenant, protecting your data from unwanted access by any other businesses or hackers.

Complete User Management

FusionAuth lets you manage individual and groups of users with role-based access control, search and segmentation, and reward/discipline tools.

Multi-Platform Deployment

FusionAuth deploys on operating system and can connect with systems on any platform, making it the only auth  you need to know.

JSON REST API

Our powerful JSON REST API provides access to all FusionAuth's features and functions. Developers can easily build a custom system to fit your needs.

MFA, SSO, Email Verification

Out of the box, FusionAuth provides today's most trusted tools for secure customer identity and authorization management.
